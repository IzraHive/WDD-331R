Setting the Stage

In 2003, CSS Zen Garden demonstrated the power of CSS for visual design. Today, Style Stage is a modern evolution of that idea.

The premise is simple: all submissions use the same HTML, styled only with CSS to create completely unique designs.

Guidelines

Here are the rules for Style Stage submissions

The Rules

  • All submissions must use the provided HTML without modifications
  • CSS must be your own original work
  • You may use CSS preprocessors like Sass
  • You may use modern CSS features
  • Design must be responsive
  • All media must be owned by you or properly licensed

How to Participate

Download the source files, create your stylesheet, and submit your work following the contribution guidelines.

Contribute

Ready to create your own Style Stage design?

  1. Download the starter files from the repository
  2. Create your unique stylesheet
  3. Test your design across different devices
  4. Submit your work via pull request
